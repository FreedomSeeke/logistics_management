# logistics_management
一个物流管理系统

#安装与使用

```bash
git clone https://github.com/FreedomSeeke/logistics_management.git
```

#创建虚拟环境：

```
python -m venv venv
```

#安装依赖
```
pip install -r requirements.txt
```
